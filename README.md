## 💬 서비스 소개

`coffeeing`는 원두와 캡슐을 추천 받을 수 있는 서비스입니다. 키워드 기반 큐레이션, 나만의 커피 취향을 분석할 수 있으며 사용자들과 피드를 통해 커뮤니케이션할 수 있습니다.
